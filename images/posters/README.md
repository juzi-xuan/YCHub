# 活动海报图片目录

本目录用于存放活动卡片的海报图片。

## 图片要求

- **推荐尺寸**: 800x600 像素（或 4:3 比例）
- **支持格式**: JPG, PNG, WebP
- **文件命名**: 使用英文小写字母和连字符

## 建议

- 可以使用活动主题相关的真实图片
- 确保图片清晰、色彩鲜明
- 建议使用专业的设计工具制作统一风格的海报
- 图片文件大小建议控制在 200KB 以内以提升加载速度
