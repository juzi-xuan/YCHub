## 📖 项目简介

**青途校园 YCHub** 是一款面向高校学生的校园活动信息聚合与服务平台，旨在整合校园内分散的学科竞赛、志愿公益、学术讲座、社团文体、招聘宣讲等各类活动信息，为学生提供一站式的活动查询、筛选、报名与管理服务。

### 🎯 核心价值

- **信息聚合**：整合分散在各渠道的活动信息，统一入口
- **便捷服务**：提供活动查询、筛选、报名、提醒一站式服务
- **高效连接**：打造学生与校园活动之间的高效连接桥梁

### 📱 页面结构

```
YCHub/
├── index.html          # 首页 - 轮播图 + 搜索
```

## 🎨 设计特色

### 视觉风格
- **主题色**：天青色（#7BA1A9）—— 取自"雨过天青云破处"
- **配色方案**：天青渐变背景 + 白色卡片 + 深色文字
- **设计理念**：国风雅致、清新简洁、信息层次清晰

### 交互体验
- 3D层叠轮播图，自动播放 + 手动切换
- 卡片悬停动效，提升交互反馈
- 响应式布局，多端适配

## 📦 快速开始

### 本地运行

```bash
# 克隆项目
git clone <repository-url>

# 进入项目目录
cd YCHub

# 使用本地服务器打开（推荐）
# Python 3
python -m http.server 8000

# 或 Node.js
npx serve .

# 然后在浏览器访问
http://localhost:8000
```

### 直接打开

由于项目使用纯前端技术，也可以直接在浏览器中打开 `index.html` 文件。
